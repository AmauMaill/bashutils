# Installation

# Scripts

Get the utils:

```bash
mkdir -p ~/bin
cd ~/bin/
git clone git@github.com:AmauMaill/bashutils.git
```

In ~/.profile, put:

```bash
export PATH="$PATH:$HOME/bin"
```

## gsbook

Use [ghostscript](https://www.ghostscript.com/) to compress PDF files.

```bash
sudo apt install ghostscript
```
